Instrukcja użytkowania

Rejestracja nowego konta

Aby zarejestrować nowe konto, wykonaj zapytanie POST na poniższym adresie URL:

POST /signup/

Przykładowe dane do przesłania:

{
        "username": "exampleuser",
        "password": "secretpassword",
        "email": "example@example.com"
        }

Logowanie

Aby zalogować się na swoje konto, wykonaj zapytania:

POST /login/

Przykładowe dane do przesłania:

{
        "username": "exampleuser",
        "password": "secretpassword"
        }

Aby pobrać informacje o użytkowniku, wykonaj zapytanie GET na poniższym adresie URL:

GET /login/

Zarządzanie produktami

Aby zarządzać produktami, wykonaj zapytania:

Wyświetl listę produktów:

GET /products/

Dodaj nowy produkt:

POST /products/

Przykładowe dane do przesłania:

{
        "name": "Example Product",
        "price": 9.99,
        "description": "This is an example product"
        }

Wyświetl, edytuj lub usuń produkt o konkretnym ID:

GET /products/{id}/
PUT /products/{id}/
DELETE /products/{id}/
